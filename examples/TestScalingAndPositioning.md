# Test Scaling and Positioning

## Scaling the chart

Scaling factor 0.5

```tracker
searchType: tag
searchTarget: exercise-pushup
folder: diary
endDate: 2021-01-31
fixedScale: 0.5
fitPanelWidth: false
line:
    title: PushUp
    yAxisLabel: Count
    lineColor: "#d65d0e"
```

